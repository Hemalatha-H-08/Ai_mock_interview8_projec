# AI Mock Interview

A full-stack AI mock interview application with React frontend and Node.js/Express backend.

## Repository

GitHub: https://github.com/Hemalatha-H-08/Ai_mock_interview8_projec

## Project structure

- `client/` — React frontend built with Vite
- `server/` — Express API server with authentication, interview history, and resume upload support

## Run locally

1. Install dependencies for both frontend and backend:

   ```bash
   cd client
   npm install
   cd ../server
   npm install
   ```

2. Start the client:

   ```bash
   cd client
   npm run dev
   ```

3. Start the server:

   ```bash
   cd server
   npm run dev
   ```

## Notes

- Configure environment variables in `server/.env` before starting the backend.
- The frontend and backend are separate applications. Run each in its own terminal.
- Update the README with your own deployment instructions if needed.
